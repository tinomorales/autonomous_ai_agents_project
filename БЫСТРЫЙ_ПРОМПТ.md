Скопируй и вставь в ChatGPT или Claude:

```
Я получил проект системы автономных ИИ-агентов для реализации. 

Проект: система ИИ-агентов работающих 24/7 без человека. Агенты автоматически генерируют задачи, пишут код по TDD, проверяют и исправляют ошибки (3 попытки, потом смена подхода).

Документация:
- ТЕХНИЧЕСКАЯ_АРХИТЕКТУРА.md - детальная архитектура всех компонентов
- Архитектура_автономных_ИИ_агентов.md - общая архитектура
- ИНСТРУКЦИЯ_ДЛЯ_РАЗРАБОТЧИКА.md - инструкция

Основные компоненты:
- Orchestrator - координация агентов
- Agent System (CEO, CFO, Developer, QA) - агенты
- Memory System (SQLite) - память
- File System Manager - редактирование файлов
- Git Manager - Git операции
- LLM Client - OpenAI/Anthropic
- Test Runner - запуск тестов

Помоги:
1. Объясни архитектуру простым языком
2. Объясни ключевые алгоритмы (непрерывный цикл CEO, самокоррекция, TDD)
3. Составь план реализации с чего начать
4. Объясни технические детали (структуры данных, потоки)
5. Помоги с первыми шагами (структура папок, первые файлы)

Будь конкретным, давай примеры кода.
```
