# Автономная система ИИ-агентов 24/7

## Описание проекта

Система полностью автономных ИИ-агентов, работающих 24/7 без участия человека. Агенты автоматически генерируют задачи, пишут код, проверяют и исправляют ошибки, принимают решения.

## Быстрый старт для разработчика

### 1. Изучи документацию

**Начни с этих документов (в порядке приоритета):**

1. **`ИНСТРУКЦИЯ_ДЛЯ_РАЗРАБОТЧИКА.md`** - начни отсюда!
2. **`docs/ТЕХНИЧЕСКАЯ_АРХИТЕКТУРА.md`** - ⭐ ГЛАВНЫЙ ДОКУМЕНТ с детальной архитектурой
3. **`docs/Архитектура_автономных_ИИ_агентов.md`** - общая архитектура
4. **`ПРОМПТ_ДЛЯ_ИЗУЧЕНИЯ_ПРОЕКТА.md`** - используй этот промпт с ChatGPT/Claude для быстрого изучения

### 2. Используй промпт для изучения

Открой `ПРОМПТ_ДЛЯ_ИЗУЧЕНИЯ_ПРОЕКТА.md` и скопируй промпт в ChatGPT или Claude. Это поможет быстро разобраться в проекте.

### 3. Понять ключевые принципы

- **TDD подход**: Тесты пишутся ПЕРВЫМИ, затем код
- **Самокоррекция**: Агенты проверяют и исправляют ошибки (3 попытки, потом смена подхода)
- **Непрерывность**: CEO генерирует задачи когда нечем заняться
- **Динамические критерии**: Агенты сами определяют критерии одобрения
- **Автономность**: Все решения без участия человека

### 4. Порядок реализации

## Технологический стек

- Python 3.11+
- OpenAI API / Anthropic API
- SQLite (для памяти)
- GitPython (для Git операций)
- FastAPI (для REST API, опционально)
- Docker (для деплоя)

## Настройка окружения

1. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создай `.env` файл:
   ```
   LLM_PROVIDER=openai
   OPENAI_API_KEY=your_key_here
   # или
   ANTHROPIC_API_KEY=your_key_here
   ```

3. Скопируй `config/config.example.yaml` в `config/config.yaml` и настрой

## Ключевые особенности

### Непрерывная работа
- Система работает 24/7 без остановки
- CEO генерирует задачи когда нечем заняться
- Автоматическое распределение задач

### TDD подход
- Тесты пишутся первыми
- Код пишется чтобы пройти тесты
- Автоматический запуск тестов

### Самокоррекция
- Агенты проверяют свои результаты
- Автоматическое исправление ошибок
- Смена подхода после 3 неудачных попыток

### Автономные решения
- Агенты принимают решения без человека
- Динамические критерии одобрения
- Многоуровневая проверка
