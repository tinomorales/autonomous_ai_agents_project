# Структура проекта для разработчика

## Текущая структура

```
autonomous_ai_agents_project/
├── docs/                          # Вся документация
│   ├── ТЕХНИЧЕСКАЯ_АРХИТЕКТУРА.md  # ⭐ ГЛАВНЫЙ ДОКУМЕНТ - начни отсюда!
│   ├── Архитектура_автономных_ИИ_агентов.md
│   ├── ОРГАНИЗАЦИОННАЯ_СТРУКТУРА_ИИ_АГЕНТОВ.md
│   ├── Архитектура_расширенная_с_орг_структурой.md
│   ├── ПЛАН_РАСШИРЕНИЯ_СИСТЕМЫ.md
│   ├── РЕЗЮМЕ_СИСТЕМЫ.md
│   ├── ИНТЕГРАЦИЯ_И_ИСПОЛЬЗОВАНИЕ.md
│   └── QUICK_START.md
├── src/                           # Здесь будет код (пока пусто)
├── config/                        # Конфигурация
│   └── config.example.yaml        # Пример конфигурации
├── README.md                      # Описание проекта
├── ИНСТРУКЦИЯ_ДЛЯ_РАЗРАБОТЧИКА.md # Инструкция для разработчика
├── requirements.txt               # Зависимости Python
└── .gitignore                     # Git ignore
```

│   │   ├── __init__.py

│   ├── deployment/                # Деплой

## Порядок создания файлов

1. Создать структуру папок в `src/`
2. Реализовать компоненты в порядке приоритета (см. ИНСТРУКЦИЯ_ДЛЯ_РАЗРАБОТЧИКА.md)
3. Добавить тесты в `tests/`
4. Настроить конфигурацию
5. Создать Dockerfile и docker-compose.yml
